# apps/go

This is a placeholder directory for the future `go` app.

How to use:

- Replace this placeholder with your Go/Next.js/other app inside `apps/go`.
- When ready, create a Vercel project and set the Root Directory to `apps/go`.

Tips:
- For Next.js, add `package.json` scripts `dev`, `build`, `start` and proper dependencies.
- For static Go web apps, ensure the build outputs to a directory Vercel can serve.
